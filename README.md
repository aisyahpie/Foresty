# Forest Management System (FMS)

A web-based Forestry Management System developed to simulate, analyze, and visualize forest data for sustainable forest management.

## Overview

The Forest Management System (FMS) was developed as part of the **Emerging Trends in Software Engineering** course. The application helps users analyze forest data, compare different cutting regimes, and generate reports through an interactive web interface.

## Features

- Forest Data Dashboard
- Stand Table Analysis
- Tree Damage Analysis
- Production Reports (Year 0 & Year 30)
- Tree Distribution Visualization
- Histogram Analysis
- Cutting Regime Comparison (45cm, 50cm, 55cm, 60cm, and 65cm)

## Technologies Used

- PHP
- HTML
- CSS
- MySQL
